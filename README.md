paramalama
==========

> Use with jQuery or Ender as $.paramalama() or standalone as paramalama().

## Install

```shell
# npm
npm install paramalama

# bower
bower install paramalama

# ender
ender add paramalama
```

## Usage

```javascript
// jQuery and Ender
$.paramalama( window.location.href );

// Standalone
paramalama( window.location.href );
```
